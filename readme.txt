/********************************************************************************
* 函数名 : IAP_Uart_Ymodem
* 功  能 : IAP功能，使用串口，采用Ymodem进行升级
* 说  明 : 基于Stm32F407ZGT6
* 入  参 : none
* 返  回 : none
* 设  计 : Shatang                    时  间 : 2020.09.17
* 修  改 : none                       时  间 : none
********************************************************************************/
1. 该IAP升级，通过超级终端进行串口传输文件，Ymodem协议
2. 文件名、类型会进行获取(与Ymodem协议有关，但是文件信息被程序内屏蔽不会显示
3. 该Bootloader进行升级需要重新上电启动
4. 支持双串口升级
5. 上电0.8s检测是否上位机是否一直发送空格，且满足15个；0.8s后检测不到(或升级完毕)进入App程序
6. 升级过程在超级终端有提示
7. 不进行CRC校验，带有加密示例(注释)
8. Bootloader程序总大小为7KB
